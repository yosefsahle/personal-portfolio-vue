# 🌐 Personal Portfolio Website (Vue.js)

A modern, responsive, and professional personal portfolio website built using **Vue.js**.
This project showcases my skills, projects, and experience in a clean and interactive way.


## ✨ Features

* ⚡ Fully responsive design (mobile, tablet, desktop)
* 🎨 Modern UI/UX with smooth animations
* 🧩 Component-based architecture (Vue)
* 📂 Projects showcase section
* 👤 About Me section
* 📬 Contact form / social links
* 🚀 Fast and optimized performance


## 🛠️ Tech Stack

* **Frontend Framework:** Vue.js
* **Styling:**  Tailwind CSS
* **Build Tool:** Vite / Vue CLI
* **Version Control:** Git & GitHub


## 📁 Project Structure

```
src/
│
├── assets/        # Images, icons, styles
├── components/    # Reusable components
├── views/         # Pages (Home, About, Projects, Contact)
├── router/        # Vue Router setup
├── App.vue        # Root component
└── main.js        # Entry point
```


## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yosefsahle/personal-portfolio-vue.git
cd personal-portfolio-vue
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```


## 🎯 Customization

You can easily customize:

* 🖼️ Images → `/assets`
* 🧑 Personal info → Update components (About.vue, Home.vue)
* 📁 Projects → Edit Projects section
* 🎨 Colors → CSS variables or Tailwind config
